# SkyCast 🌤️

A live weather app built with React + OpenWeatherMap API.

## Features
- 🌍 Search any city worldwide
- 📍 GPS-based location detection
- 🌡️ Hourly + 5-day forecast
- 🗺️ Interactive weather map
- ⭐ Favourites & recent searches
- 🌙 Dark / Light theme
- ⚠️ Live weather alerts


## Tech Stack
React • Custom Hooks • REST API • CSS Grid • Leaflet Maps

## Setup
```bash
git clone https://github.com/vishss11/skycast
cd skycast
npm install
echo "REACT_APP_WEATHER_API_KEY=your_key_here" > .env
npm start
```
